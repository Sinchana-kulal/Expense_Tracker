Project Overview :-
The Expense Tracker App is a simple React application that allows users to track their income and expenses, calculate the current balance, and manage a transaction history.
This project is built to understand React fundamentals, especially state management using hooks.

Objectives:-
•	Track daily income and expenses
•	Calculate total income, total expenses, and balance
•	Practice React concepts like useState

Features:-
•	Add income and expense transactions
•	Delete transactions from history
•	Automatic calculation of:
•	Total Income
•	Total Expense
•	Balance
•	Transaction history display
•	Simple and user-friendly UI

Technologies Used:-
•	React.js
•	JavaScript (ES6)
•	CSS
•	React Hooks (useState)
